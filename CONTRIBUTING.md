# Contributing

## Scope

This is an open-review collection, not a journal. Feedback may identify a
gap, a counterexample, a citation, a clearer proof, or a reproducibility
problem. It must not be read as a publication decision or a request for
automatic coauthorship.

## Review Requests

Use the supplied issue forms and include a paper version and a precise
location. For a proof issue, quote the theorem, lemma, equation, or algorithm
step and explain the missing implication. For a counterexample, include a
minimal input and enough detail for another reader to reproduce it.

Please do not open an issue that only says the whole paper needs checking.
The valuable unit of review is a falsifiable or verifiable mathematical claim.

## Revisions

The LaTeX source inside a paper version is its only editable manuscript
source. Accepted changes produce a new version directory, a new PDF, updated
evidence where relevant, and a release note that links the review finding to
the change. Existing version directories are frozen.

## Attribution And Privacy

Contributors retain credit for their own feedback. Do not post private contact
details, institutional credentials, or unpublished manuscripts without
permission. A useful correction will be acknowledged in a later release only
with the contributor's explicit consent.
